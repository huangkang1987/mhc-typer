# mhc-typer
MHC-Typer V1.1

Assigning alleles in multilocus amplification. 

Requirements:
Windows platform, and .net framework 2.0 runtime library.

New Features:
V1.1: Considering CNV, heterozygote excess, sharing of identical alleles between loci.
